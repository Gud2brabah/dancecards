About

  • The dancecard "app" server side is written in Python using the Flask framework
  • Browser-side scripting handled with custom JavaScript and CSS hackery
  • The underlying database uses sqlite3
  • Hosting currently using nginx on an AWS Lightsail instance
  • Available wherever modern browsers are found
